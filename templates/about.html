About!

The Sentiment Analysis for YouTube Comments using AI project is designed to classify and analyze the sentiments expressed in YouTube video comments using Natural Language Processing (NLP) and machine learning techniques. The system collects trending YouTube comments data from sources like Kaggle and preprocesses them by handling missing values, removing duplicates, and performing tasks such as tokenization, lemmatization, and stopword removal. Sentiment scores are generated using NLP, which classifies comments as positive, negative, or neutral. To achieve high accuracy, advanced AI models such as BERT, LSTM, and GRU are integrated for sentiment classification. The model is trained and evaluated using metrics such as accuracy, precision, recall, and F1-score to ensure robust performance. Once trained, the model is deployed within a web application where users can input a YouTube video URL, retrieve the associated comments, and visualize the sentiment distribution in the form of a graph displaying the percentage of positive, negative, and neutral comments. The project provides a user-friendly interface that enables easy sentiment analysis, making it useful for content creators, marketers, and casual users to gauge audience sentiment effectively. By automating the process of comment analysis, this system offers a scalable and efficient solution for understanding public opinion on YouTube videos
